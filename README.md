# Sistem Manajemen Stok dan Transaksi Gudang Pakan Ternak

Sistem manajemen gudang yang komprehensif untuk mengelola stok dan transaksi pakan ternak secara efisien dan real-time.

## 📋 Deskripsi Proyek

Sistem ini dirancang untuk membantu pengelolaan inventori gudang pakan ternak dengan fitur-fitur utama:

- ✅ Pencatatan stok pakan ternak
- ✅ Manajemen transaksi masuk dan keluar
- ✅ Pelaporan dan analisis data stok
- ✅ Visualisasi data penjualan dan stok
- ✅ Antarmuka pengguna yang user-friendly

## 🛠️ Tech Stack

- **Language**: Python 🐍
- **Editor**: Visual Studio Code 💻
- **Version Control**: Git & GitHub

### Libraries:
- **pandas** - Manipulasi dan analisis data
- **matplotlib.pyplot** - Visualisasi data grafis
- **colorama** - Warna terminal untuk UI yang lebih menarik
- **tabulate** - Pemformatan tabel data yang rapi

## 📦 Instalasi Dependencies

```bash
pip install pandas matplotlib colorama tabulate
```

## 🚀 Cara Menggunakan

1. **Clone Repository**
   ```bash
   git clone https://github.com/Alfauzi-S/Sistem_Manajemen_Stok_dan_Transaksi_Gudang_Pakan_Ternak.git
   ```

2. **Navigasi ke Direktori Proyek**
   ```bash
   cd Sistem_Manajemen_Stok_dan_Transaksi_Gudang_Pakan_Ternak
   ```

3. **Install Dependencies**
   ```bash
   pip install pip install pandas matplotlib colorama tabulate
   ```

4. **Jalankan Aplikasi**
   ```bash
   python main.py
   ```

## 📚 Fitur Utama

### Manajemen Stok
- Tambah stok pakan baru
- Update ketersediaan stok
- Hapus item stok

### Transaksi
- Catat transaksi masuk (pembelian)
- Catat transaksi keluar (penjualan)
- Lihat riwayat transaksi

### Laporan & Analisis
- Laporan stok real-time
- Visualisasi grafik penjualan
- Analisis data dengan pandas
- Export data ke format tabel
